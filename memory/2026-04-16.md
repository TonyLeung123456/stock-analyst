## 🔧 重大Bug修复：JavaScript转义符导致按钮全部失效（19:47）

**问题**：所有按钮（除测试按钮）点击无响应，`runScan is not a function`

**根本原因**：
1. HTML中20处反引号被错误写成 `\``（反斜杠+反引号）
2. 这导致JavaScript语法错误，整个 `<script>` 块失效
3. `runScan()` 等所有函数未定义
4. 测试按钮能工作是因为它是内联 `onclick`（不依赖外部JS函数定义），且用的是单行变量赋值语句（在语法错误前就已解析）

**排查方法**：
1. `node --check <script>.js` → 精确定位语法错误
2. 浏览器 `new Function(js_code)` → 显示错误位置
3. `document.querySelectorAll('script')` → 确认只有一个内联脚本
4. `typeof window.runScan` → 确认函数是否定义

**修复**：
```python
# 把所有 \` 替换为 `
content.replace('\\`', '`')
```

**教训**：
- Python raw string (r"""..."""") 中写JS模板字符串，不要用 \` 转义反引号，直接用 ` 即可
- 函数定义检查：`typeof window.fnName === 'function'`
- 测试按钮用内联 `onclick` 写法是因为它是独立简单的逻辑；复杂函数调用必须正确定义
