### 浏览器缓存导致 JS 报错（进行中）
**现象**：Chrome/Safari 访问 http://localhost:7878 时按钮无反应，Console 报错：
```
Uncaught SyntaxError: Unexpected string
Uncaught ReferenceError: switchTab is not defined
```

**分析**：
- 服务端 served HTML 确认正确（curl 测试通过）
- 无痕模式可用（说明代码本身没问题）
- **结论**：浏览器缓存了旧版 HTML/JS

**解决方向**：
- 用户需在 DevTools Network 面板勾选 "Disable cache" 后强制刷新
- 或完全关闭浏览器再重新打开
- 服务端添加 Cache-Control 头防止缓存

**关键代码位置**：
- `switchTab` 定义：HTML 中所有 `onclick="switchTab(...)"` 均正确
- 第 3002 行附近 showAnalysis onclick 包含 enrichment 逻辑（名称/板块显示）
