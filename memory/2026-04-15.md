## 二、地缘政治与宏观变量分析

### 🔴 中美关系：关税战持续升级，但边际恶化速度趋缓

**核心事件（4月7-11日当周）：**
- 中美互相加征报复性关税，税率维持在**145%（美对中）/ 125%（中对美）**高位
- 美国对华加征145%关税涉及：半导体设备、AI芯片、消费电子、光伏产品
- 中国反制125%关税涉及：农产品、能源、汽车

**市场影响路径：**
```
关税升级 → 出口预期下修 → A股电子/光伏/半导体Q2盈利预测下调
         → 进口替代逻辑强化 → 国产替代板块（半导体设备/软件）获得对冲资金流入
         → 北向资金持续观望（4月整月净流出压力较大）
```

**重要信号**：4月11日（周五）传出双方在瑞士进行非正式接触的消息，关税进一步升级的边际恶化速度可能趋缓，**市场已开始Price in最差情景**

**出口数据影响（4月11日发布）**：
- 3月出口数据整体低于预期，但市场已提前反应
- 对美出口同比下滑超预期，但其他市场（东盟、欧盟）部分对冲

### 🟡 美伊谈判：地缘风险降温信号

**核心事件（4月11日周六）**：
- 美伊在维也纳恢复间接谈判（2020年来首次）
