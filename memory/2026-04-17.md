### 问题：页面按键失灵，底部CSV导出按钮点击无反应

**排查过程：**
1. 浏览器工具无法连接 Chrome DevTools（端口9222连接超时）
2. 无法截图看视觉效果
3. 通过 Python 源码分析，发现 HTML 里内嵌的 JS 有两处语法错误

**错误1：** `onclick="showStockDetail(''+s.code+'')"` — JS 引号嵌套语法错误
- 位置：股票搜索结果 onclick
- 修复：`onclick="showStockDetail(s.code)"` — 直接传 JS 变量

**错误2：** `.join('\n')` — Python 源码中 `'\n'` 是真实换行符（1字节 0x0A），不是字符串 `'\\n'`
- 位置：CSV 导出函数 rows.map().join()
- 影响：HTML 中嵌入的 JS 字符串被换行符截断，产生跨行语法错误
- 修复：用 `&& newline = String.fromCharCode(10)` 临时变量方案

### 修复后重启

```bash
# 删除旧 .pyc 缓存
find daily_report_app/ -name "*.pyc" -delete
rm -rf daily_report_app/__pycache__/

# 重启 app
```

### 重要教训（补）：JavaScript模板字符串与Python raw string（2026-04-17 晚间21:15）

**核心教训：**
- Python raw string（r"""..."""）里写 JavaScript 模板字符串时，反引号直接写 `` ` `` 就行，不要加反斜杠转义
- 加了反而变成语法错误，整段 JS 就全崩了
- 后来改用普通 Python 字符串（不用 r 前缀）写 JS 内容，双引号内不需要对 `${}` 转义

**本次问题：** 
- `sortTable` 函数的 `onclick` handler 中，`onclick="sortTable(\''+c.key+'')"` JS 引号嵌套语法错误
- 修复：改用 JS 模板字符串 `` `onclick="sortTable('${c.key}')"` ``
- 但写入 Python 字符串时，`${}` 会触发 Python f-string 解析，需要用 `{{` `}}` 或直接用单引号包裹的 JS 字符串

**标准排查流程：**

### 补充：sortTable onclick 模板字符串 bug（2026-04-17 21:30）

**问题：** 在 Python 字符串里写 JavaScript 模板字符串时，onclick 属性中不能写 `${c.key}` 这种模板占位符，因为 HTML onclick 属性值不是 JS 模板字符串上下文。

**错误写法：**
```python
f"<th ... onclick=\"sortTable('{c.key}')\">"  # c.key 是 JS 变量，不会被替换
```

**正确写法：** 用 `this` + `data-key` 属性，让 sortTable 运行时自己读取：
```python
f'<th data-key="{c.key}" onclick="sortTable(this)">'
# sortTable 函数内部:
const key = this.getAttribute('data-key');
```

**教训：** onclick 属性里不能放 JS 变量引用，要通过 DOM 属性传参。

### 重要教训（最终版）：函数作用域与HTML onclick（2026-04-17 21:40）

**问题：** 标签页按钮点击无效，hover有变化但点击无反应

**根本原因：** 
- HTML按钮有`onclick="switchTab(this)"`属性
- 但`switchTab`函数是在`document.querySelectorAll('.tab').forEach(btn => {...})`的`addEventListener`回调里定义的
- `onclick`需要`window.switchTab`（全局函数），但函数在局部作用域，按钮找不到，所以点击失败

**修复方案：**
```javascript
// 错误（局部作用域）：
document.querySelectorAll('.tab').forEach(btn => {
  btn.addEventListener('click', () => {
    switchTab(btn);  // ← 这是局部函数，HTML onclick 找不到
  });
});

// 正确（全局函数）：
function switchTab(domEl) {  // ← 定义在全局，window.switchTab 可访问
  ...;
}
```

**教训：** HTML 内联 `onclick="func()"` 需要函数在 `window` 全局作用域。`addEventListener` 里定义的函数是局部作用域，HTML onclick 访问不到。
